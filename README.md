# Simple React

React example without setting.

# Start

```sh
npm install
npm start
```

http://localhost:3000/

> If you want to using `python`(Prerequisites installed `python`)

```sh
python -m SimpleHTTPServer 3000
```

check [index.html](index.html) in your browser.

http://localhost:3000/
